# Dotnet Library Setup Profile

Use this profile when a repository produces one or more reusable .NET libraries.

Common active layers:

```text
README.md
AGENTS.md
docs/TERMINOLOGY.md
docs/SPECS.md
docs/ENGINEERING.md
docs/MILESTONES.md
```

When public packages are active, add:

```text
docs/PUBLIC-DOCS.md
public-docs/
```
